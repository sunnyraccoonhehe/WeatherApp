Weather App - это React-приложение, которое предоставляет актуальную информацию о погоде в любой точке мира. Приложение отличается красивым адаптивным дизайном, динамической сменой фона в зависимости от погодных условий и интуитивно понятным интерфейсом.

Особенности

- **Динамический фон** - автоматически меняется в зависимости от типа погоды (солнечно, облачно, дождь, снег, гроза)
- **Умный поиск городов** - автодополнение при вводе названия города
- **Детальная информация**:
  - Текущая температура
  - Влажность воздуха
  - Скорость ветра
  - Атмосферное давление
  - Время восхода и заката
  - Продолжительность светового дня
- **Прогноз на неделю** - информация о погоде на ближайшие 6 дней
- **Адаптивный дизайн** - отлично выглядит на всех устройствах


Технологии

- **React** - библиотека для построения пользовательского интерфейса
- **CSS3** - стилизация и анимации
- **Axios** - для HTTP-запросов к API
- **Font Awesome** - иконки для интерфейса
- **Open-Meteo API** - бесплатный API для получения погоды
- **Open-Meteo Geocoding API** - поиск городов и получение координат

Установка

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/your-username/weather-app.git
cd weather-app
```
2. **Установите зависимости**
```bash
npm install
```
3. **Запустите приложение**
```bash
npm start
```
4. **Для создания сборки выполните команду**
```bash
npm run build
```
